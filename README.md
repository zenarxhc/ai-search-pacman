## 📖 What is ai-search-pacman?

ai-search-pacman is a simple program that lets you watch how a computer solves Pacman mazes. It uses classic search methods like DFS (Depth-First Search), BFS (Breadth-First Search), UCS (Uniform Cost Search), and A* (A-star). It also uses smart guesses (heuristics) to find solutions faster.

This tool is made by UC Berkeley and is a good way to see how pathfinding and problem-solving work in games with AI. You don’t need to know how to code to use this app. Just download and watch Pacman find a path through the maze step by step.
